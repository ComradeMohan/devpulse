# **App Name**: DevPulse

## Core Features:

- GitHub Summary: Display total commits, public repos, stars earned, forks, and follower count. Includes animated stat cards.
- Interactive Charts: Show top programming languages (pie chart), commits per month (bar chart), yearly contribution graph, and activity heatmap.
- Achievement Badges: Show GitHub-style achievement badges in large, glossy cards. Achievements include Arctic Code Vault Contributor, Pull Shark, YOLO, Galaxy Brain, Starstruck, and Developer Program Member.
- Repository Explorer: Allows sorting by stars, forks, recent activity, and language. Provides search functionality to locate repos.
- Contribution Analysis Tool: Use a generative AI model to provide personalized insights and suggestions to improve your contribution habits, such as optimal times for commits or potential collaboration opportunities. It acts as a tool by reasoning over the data and incorporates these recommendations based on context.

## Style Guidelines:

- Primary color: Neon Purple (#9400D3) to evoke a modern, developer-focused aesthetic.
- Background color: Dark Black (#121212) for a sleek, dark theme.
- Accent color: Electric Blue (#7DF9FF) for highlighting interactive elements and glowing effects.
- Headline font: 'Space Grotesk' sans-serif for headlines and 'Inter' sans-serif for body text for a computerized, modern feel.
- Use simple, line-based icons with a neon blue glow to match the developer aesthetic.
- Employ a modern, card-based layout with rounded corners and glass-morphism effects.
- Use subtle fade-in animations and hover effects to provide a smooth, interactive user experience.
